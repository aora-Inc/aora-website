## 1️⃣ Finance - PUC Token

### コンセプト
aoraロゴの「4つの円」をモチーフに、PUC（Personal Universal Credit）を表現

### ビジュアル構成
```
◯ ◯ ◯ ⚫
```
- 3つの白い円（カーボンクレジット）
- 1つの青グラデーション円（PUC）
- 最後の円が光る（Glow Effect）

### アニメーション
- **常時**: 青い円がパルスグロー（2秒サイクル）
- **ホバー時**: 4つの円が順番に浮き上がる（0.05s 間隔）

### 実装ファイル
- HTML: `.icon-finance` > `.puc-token` (x4)
- CSS: Line 676-717

## 2️⃣ IP Development - Pucreキャラクター

### コンセプト
Pucreアプリの"キャラクター"を象徴する、親しみやすい顔のデザイン

### ビジュアル構成
```
😊
```
- ピンクグラデーションの円形ベース
- 2つの目（白い円）
- 笑顔の口

### アニメーション
- **常時**: バウンスアニメーション（3秒サイクル、上下に移動）
- **目**: 瞬きアニメーション（4秒サイクル）
- **グロー**: ピンクの柔らかいグロー

### 実装ファイル
- HTML: `.icon-character` > `.character-face`
- CSS: Line 719-778

---

## 3️⃣ Creative - 再生ボタン ＋ サウンドウェーブ

### コンセプト
動画・楽曲制作を表現する、メディアコンテンツの象徴

### ビジュアル構成
```
▶ |||
```
- 円形の再生ボタン（中央に三角形）
- 右側にサウンドウェーブ（3本の縦線）

### アニメーション
- **常時**: サウンドウェーブが上下に波打つ（1.5秒サイクル、各0.15s 遅延）
- **ホバー時**: 再生ボタンが青グラデーションに変化

### 実装ファイル
- HTML: `.icon-creative` > `.play-button` + `.sound-waves`
- CSS: Line 780-861

---

## 4️⃣ Gamification - ゲームコントローラー

### コンセプト
Pucreアプリの「遊ぶように環境貢献」を表現するゲームコントローラー

### ビジュアル構成
```
[十字キー] [◯◯]
```
- コントローラー本体（角丸の矩形）
- 左側: 十字キー（D-Pad）
- 右側: 2つのボタン

### アニメーション
- **ホバー時**: 
  - 背景が紫グラデーションに変化
  - 全体が1.05倍に拡大

### 実装ファイル
- HTML: `.icon-game` > `.game-controller`
- CSS: Line 863-928

## 🎯 技術仕様

### 共通設定
- **サイズ**: 80-100px
- **実装方法**: CSS のみ（SVG / 画像ファイル不要）
- **パフォーマンス**: GPU加速（transform）
- **アクセシビリティ**: セマンティックHTML + ARIA

### カラーパレット
| 要素 | カラー |
|------|--------|
| Finance | Blue Gradient: `linear-gradient(135deg, #00D4FF 0%, #0088CC 100%)` |
| IP | Game Gradient: `linear-gradient(135deg, #EC4899 0%, #F472B6 100%)` |
| Creative | Blue Light: `#00D4FF` |
| Gamification | Purple: `#8B5CF6` |
